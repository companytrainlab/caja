# TrainLab · el landing de la caja

La página que abre el móvil al acercar el tag NFC de una caja TrainLab:

```
https://companytrainlab.github.io/caja/#CODIGO
```

Lee el código del `#hash`, pide los datos a la nube de TrainLab y enseña
la guía de primeros pasos del producto correcto, el botón que abre la app
y el estado de la garantía. Una sola página, sin dependencias.

Este repositorio es público solo porque GitHub Pages lo necesita: aquí no
hay secretos ni lógica de negocio — los datos y la garantía viven en el
servidor. © TrainLab. Código a la vista no significa licencia de uso.

> ⚠️ **NUNCA renombrar ni mover este repositorio.** Hay tags NFC físicos
> grabados y BLOQUEADOS con esta URL: renombrarlo los mataría todos.
> Conectar un dominio propio sí es seguro (github.io redirige solo).
